请选择 .codex 目录

继续后会打开系统目录选择器，请选择包含 sessions 目录的 .codex 根目录。下面把三个系统的常见路径和隐藏目录打开方式都写在这里了。

常见路径
目录内应包含：sessions/、archived_sessions/、session_index.jsonl、state_5.sqlite。如果看不到 .codex，通常是因为它是隐藏目录。

MACOS

~/.codex

Finder 里按 Shift + Command + . 显示隐藏文件；或者在“前往文件夹”中直接输入 ~/.codex。

WINDOWS

%USERPROFILE%\\.codex

资源管理器地址栏可直接输入上面的路径；如果手动找，请先在“查看”里开启“隐藏的项目”。

LINUX

~/.codex

文件管理器里通常按 Ctrl + H 显示隐藏文件；也可以在路径栏直接输入 ~/.codex。

如果系统目录选择器不方便选隐藏目录，也可以点主页上的“目录选择降级”，直接把整个 .codex 目录拖进去。
